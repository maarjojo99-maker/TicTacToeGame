# Tic-Tac-Toe Game

A classic two-player Tic-Tac-Toe game built with React.

## Features
- Two-player mode with customizable names.
- Dynamic game board.
- Win detection and draw handling.
- Game log tracking move history.
- Restart functionality.

## Core Concepts Covered
- Lifting state up.
- Managing complex state (arrays/objects).
- Derived state for game logic.
- Conditional rendering.

## Getting Started
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev`.
